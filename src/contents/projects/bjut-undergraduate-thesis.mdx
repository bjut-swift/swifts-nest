## 这是什么？

参照《北京工业大学本科学位论文写作规范》编写的本科毕业论文 LaTeX 模板，基于 [hw401/BJUT-Undergraduate-Thesis](https://github.com/hw401/BJUT-Undergraduate-Thesis) 做了结构调整，旨在建立一个简单易用的本科学位论文模板。

# 环境要求

| 组件       | 说明                                                                  |
| ---------- | --------------------------------------------------------------------- |
| TeX 发行版 | TeX Live（推荐 2023 及以上）                                          |
| 编译器     | XeLaTeX（必须，不可用 pdfLaTeX 或 LuaLaTeX）                          |
| 编辑器     | 推荐使用 TeXPage；不建议使用 Overleaf，可能因项目规模较大导致编译超时 |

# 使用方式

1. 把整个项目打个 `.zip` 压缩包
2. 扔进 TeXPage 选 Upload Project 创建项目
