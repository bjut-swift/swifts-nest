## 🤝 贡献方式

### 💭 如果你只想分享经验：

请在仓库中的 `docs` 文件夹中的对应目录下新建你的经验分享文档，文件名格式为 `届-专业-姓名(可选择网名)`，建议使用 `.md` 格式以方便阅读。

> 注意：由于本仓库未启用 `git-lfs`，因此请不要上传单个超过 100M 的文件，否则 commit 无效。本项目尽可能只做收录链接功能，不收录资源，贡献请直接在 README 中相关区域添加。

### 📦 如果你想分享学习资源：

直接在 README.md 中的表格对应位置填写你的信息和资源链接即可。二进制文件、笔记等资源推荐自己建仓库存储，同时也欢迎到 <a href="https://github.com/bjut-swift/BJUT-Helper" target="_blank">北京工业大学课程资源</a> 参与贡献。

## ⚖️ 免责声明

本仓库所有代码、文档、实验报告等资源仅供学习交流参考使用。

**禁止直接复制、修改后作为个人作业/实验报告提交。**
